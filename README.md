# Bipana Cafe

Welcome to the Bipana Cafe repository!  
This project is a digital showcase of Bipana Cafe, blending design and functionality to create a welcoming online presence.

## Features
- Interactive menu
- Cafe story and gallery
- Responsive layout
- Easy customization

## Getting Started
1. Clone the repo
2. Install dependencies
3. Run `npm start` to launch locally
